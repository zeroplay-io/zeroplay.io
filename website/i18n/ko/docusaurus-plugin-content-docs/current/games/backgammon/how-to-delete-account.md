# 계정 삭제 방법

**Backgammon - Fair Board Games**에서 계정을 삭제하려면 다음 단계를 따르십시오:

---

### 1단계: 설정 열기

게임을 실행하고 오른쪽 상단의 *설정* 버튼을 탭합니다。

![설정](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/1.jpg#landscape)

### 2단계: 삭제 옵션 찾기

설정 창 하단까지 스크롤하고 <span className="danger-text">계정 삭제</span> 버튼을 탭합니다。

![계정 삭제](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/2.jpg#landscape)

### 3단계: 작업 확인

화면에 표시된 확인 코드를 입력한 다음 *삭제* 버튼을 탭하여 계속 진행합니다。

![삭제 확인](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/3.jpg#landscape)

### 4단계: 최종 카운트다운

삭제 프로세스는 10초 카운트다운으로 시작됩니다. 이 시간 동안 *삭제 취소*를 탭하여 취소할 수 있습니다。

![계정 삭제됨](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/4.jpg#landscape)

---

### ⚠️ 중요 사항

- **영구 작업** — 모든 게임 데이터가 영구적으로 삭제됩니다
- **복구 불가** — 카운트다운이 끝난 후에는 이 작업을 취소할 수 없습니다
