# 계정 삭제 방법

**ZeroPlay** 게임에서 계정을 삭제하려면 다음 단계를 따르십시오:

---

### 1단계: 프로필 액세스

하단 내비게이션 바의 *마이페이지* 탭을 탭하여 프로필 페이지에 액세스합니다。

![프로필](https://dl.zeroplay.cc/metadata/how-to-delete-account/1.jpg)

### 2단계: 삭제 옵션 찾기

<span className="danger-text">위험 구역</span> 섹션까지 스크롤하고 <span className="danger-text">계정 삭제</span> 버튼을 탭합니다。

![계정 삭제](https://dl.zeroplay.cc/metadata/how-to-delete-account/2.jpg)

### 3단계: 삭제 확인

경고를 검토하고 *계속*을 탭하여 계정 삭제를 진행합니다。

![삭제 확인](https://dl.zeroplay.cc/metadata/how-to-delete-account/3.jpg)

### 4단계: 최종 카운트다운

삭제 프로세스는 10초 카운트다운으로 시작됩니다. 이 시간 동안 *삭제 취소*를 탭하여 취소할 수 있습니다。

![계정 삭제 진행 중](https://dl.zeroplay.cc/metadata/how-to-delete-account/4.jpg)

---

### ⚠️ 중요 사항

- **영구 작업** — 모든 게임 데이터가 영구적으로 삭제됩니다
- **복구 불가** — 카운트다운이 끝난 후에는 이 작업을 취소할 수 없습니다
