# 개인정보 보호정책

## 1. 수집하는 정보

1.1. 귀하가 제공하는 정보:
- 계정 등록 정보
- 프로필 정보(아바타, 사용자 이름)
- 구매 결제 정보
- 당사와의 커뮤니케이션

1.2. 제3자 플랫폼의 정보:
- 제3자 서비스를 통해 로그인할 때의 프로필 정보
- 계정 인증 정보

1.3. 자동으로 수집되는 정보:
- 기기 정보
- 사용 데이터
- IP 주소 및 위치 데이터
- 게임 진행 상황 및 환경설정

## 2. 정보 사용 방법

2.1. 수집된 정보는 다음 목적으로 사용됩니다:
- 서비스 제공 및 개선
- 거래 처리
- 게임 경험 개인화
- 서비스에 관한 커뮤니케이션
- 약관 준수 보장
- 사기 및 보안 문제 방지

## 3. 정보 공유

3.1. 다음과 정보를 공유할 수 있습니다:
- 서비스 운영을 지원하는 서비스 제공업체
- 다른 플레이어(공개 프로필 정보에 한정)
- 법률에서 요구하는 경우 법 집행 기관
- 비즈니스 거래와 관련된 제3자

## 4. 데이터 보안

4.1. 귀하의 정보를 보호하기 위해 합리적인 보안 조치를 시행합니다。

4.2. 인터넷을 통한 전송 방법 중 100% 안전한 것은 없습니다。

## 5. 어린이 개인정보 보호

5.1. 연령 요구 사항은 게임마다 다릅니다. 각 게임의 최소 연령 요구 사항 미만의 사용자로부터 고의로 개인 정보를 수집하지 않습니다。

5.2. 부모/보호자는 자녀의 개인 정보를 검토, 수정 또는 삭제하기 위해 당사에 문의할 수 있습니다。

## 6. 국제 데이터 전송

6.1. 귀하의 정보는 귀하의 국가 이외의 국가로 전송되어 처리될 수 있습니다。

6.2. 국제 전송에 대해 적절한 보호 조치를 취하고 있습니다。

## 7. 귀하의 권리 및 선택

7.1. 귀하는 다음과 같은 권리를 가질 수 있습니다:
- 개인 정보 액세스
- 부정확한 정보 수정
- 정보 삭제
- 처리에 대한 반대
- 데이터 내보내기

## 8. 개인정보 보호정책 변경

8.1. 본 개인정보 보호정책을 정기적으로 업데이트할 수 있습니다. 서비스의 지속적인 사용은 변경 사항의 수락을 구성합니다。

## 9. 문의하기

본 개인정보 보호정책에 대한 질문이 있으시면 [support@zeroplay.io](mailto:support@zeroplay.io)로 문의해 주시기 바랍니다。

## 10. 쿠키 정책

10.1. 다음 목적으로 쿠키 및 유사한 기술을 사용합니다:
- 환경설정 기억
- 서비스 사용 방식 이해
- 사용자 경험 개선
- 타겟 광고 제공

10.2. 브라우저 환경설정을 통해 쿠키 설정을 제어할 수 있습니다。
