# So löschen Sie Ihr Konto

Wenn Sie Ihr Konto aus einem **ZeroPlay**-Spiel löschen möchten, befolgen Sie diese Schritte:
<!-- Button labels remain in English to match UI screenshots: Me, DANGER ZONE, Delete Account, Continue, Cancel Deletion. -->

---

### Schritt 1: Greifen Sie auf Ihr Profil zu

Tippen Sie auf die Registerkarte *Me* in der unteren Navigationsleiste, um auf Ihre Profilseite zuzugreifen.

![Profil](https://dl.zeroplay.cc/metadata/how-to-delete-account/1.jpg)

### Schritt 2: Finden Sie die Löschoption

Scrollen Sie nach unten zum Abschnitt <span className="danger-text">DANGER ZONE</span> und tippen Sie auf die Schaltfläche <span className="danger-text">Delete Account</span>.

![Konto löschen](https://dl.zeroplay.cc/metadata/how-to-delete-account/2.jpg)

### Schritt 3: Bestätigen Sie die Löschung

Überprüfen Sie die Warnung und tippen Sie auf *Continue*, um mit der Kontolöschung fortzufahren.

![Löschung bestätigen](https://dl.zeroplay.cc/metadata/how-to-delete-account/3.jpg)

### Schritt 4: Finaler Countdown

Der Löschvorgang beginnt mit einem 10-Sekunden-Countdown. Sie können während dieser Zeit noch abbrechen, indem Sie auf *Cancel Deletion* tippen.

![Kontolöschung läuft](https://dl.zeroplay.cc/metadata/how-to-delete-account/4.jpg)

---

### ⚠️ Wichtige Hinweise

- **Dauerhafte Aktion** — Alle Ihre Spieldaten werden dauerhaft gelöscht
- **Keine Wiederherstellung** — Diese Aktion kann nach Ablauf des Countdowns nicht rückgängig gemacht werden
