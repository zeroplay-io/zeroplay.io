# So löschen Sie Ihr Konto

Wenn Sie Ihr Konto aus **Backgammon - Fair Board Games** löschen möchten, befolgen Sie diese Schritte:

---

### Schritt 1: Einstellungen öffnen

Starten Sie das Spiel und tippen Sie auf die Schaltfläche *Einstellungen* in der oberen rechten Ecke.

![Einstellungen](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/1.jpg#landscape)

### Schritt 2: Finden Sie die Löschoption

Scrollen Sie zum unteren Rand des Einstellungsfensters und tippen Sie auf die Schaltfläche <span className="danger-text">Konto löschen</span>.

![Konto löschen](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/2.jpg#landscape)

### Schritt 3: Überprüfen Sie Ihre Aktion

Geben Sie den auf dem Bildschirm angezeigten Bestätigungscode ein und tippen Sie dann auf die Schaltfläche *Löschen*, um fortzufahren.

![Löschung bestätigen](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/3.jpg#landscape)

### Schritt 4: Finaler Countdown

Der Löschvorgang beginnt mit einem 10-Sekunden-Countdown. Sie können während dieser Zeit noch abbrechen, indem Sie auf *Löschung abbrechen* tippen.

![Konto gelöscht](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/4.jpg#landscape)

---

### ⚠️ Wichtige Hinweise

- **Dauerhafte Aktion** — Alle Ihre Spieldaten werden dauerhaft gelöscht
- **Keine Wiederherstellung** — Diese Aktion kann nach Ablauf des Countdowns nicht rückgängig gemacht werden
