# Tilin Poistaminen

Jos haluat poistaa tilisi mistä tahansa **ZeroPlay** pelistä, seuraa näitä ohjeita:
<!-- Button labels remain in English to match UI screenshots: Me, DANGER ZONE, Delete Account, Continue, Cancel Deletion. -->

---

### Vaihe 1: Siirry Profiiliisi

Napauta *Me* välilehteä alareunassa olevassa navigointipalkissa päästäksesi profiilisivullesi.

![Profile](https://dl.zeroplay.cc/metadata/how-to-delete-account/1.jpg)

### Vaihe 2: Etsi Poisto-vaihtoehto

Vieritä alas <span className="danger-text">DANGER ZONE</span> osioon ja napauta <span className="danger-text">Delete Account</span> painiketta.

![Delete Account](https://dl.zeroplay.cc/metadata/how-to-delete-account/2.jpg)

### Vaihe 3: Vahvista Poisto

Tarkista varoitus ja napauta *Continue* jatkaaksesi tilin poistoa.

![Confirm Deletion](https://dl.zeroplay.cc/metadata/how-to-delete-account/3.jpg)

### Vaihe 4: Viimeinen Lähtölaskenta

Poistoprosessi alkaa 10 sekunnin lähtölaskennalla. Voit vielä peruuttaa napauttamalla *Cancel Deletion* tämän ajan kuluessa.

![Account Deletion in Progress](https://dl.zeroplay.cc/metadata/how-to-delete-account/4.jpg)

---

### ⚠️ Tärkeää Huomioitavaa

- **Pysyvä Toimenpide** — Kaikki pelitietosi poistetaan pysyvästi
- **Ei Palautusta** — Tätä toimintoa ei voi perua lähtölaskennan päätyttyä
