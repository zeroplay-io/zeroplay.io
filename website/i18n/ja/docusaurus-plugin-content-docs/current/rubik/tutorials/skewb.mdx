# スキューブの解き方

**サラの初心者向けメソッド**を使ってスキューブを解く方法を学びましょう。シンプルで効率的な初心者向けアプローチです。スキューブは角を中心に回転するパズルで、各ムーブは1つの角と隣接する3つの面を回転させます。たった1つのコアアルゴリズム「スレッジハンマー」を使って3ステップで解きます。

## 回転記号

標準的なルービックキューブとは異なり、スキューブは面ではなく**角**を中心に回転します。各ムーブは1つの角と隣接する3つのセンターピースを回転させます：

- **R** = 右前の角を時計回り &nbsp;&nbsp; **R'** = 右前の角を反時計回り
- **L** = 左前の角を時計回り &nbsp;&nbsp; **L'** = 左前の角を反時計回り
- **U** = 上右前の角を時計回り &nbsp;&nbsp; **U'** = 上右前の角を反時計回り
- **B** = 右後ろの角を時計回り &nbsp;&nbsp; **B'** = 右後ろの角を反時計回り

文字だけなら**時計回り**の回転です。**'** を付けると**反時計回り**です。

すべてのステップで、スキューブを**白いセンターを下**にして持ちます。

---

## ステップ1：1つの面を揃える

下面に完全な**白い面**を作ります。白いセンターの周りの4つの角すべてが正しい位置に、白が下を向いた状態で配置され、隣接する側面の色も一致する必要があります。

このステップは半直感的です。重要なツールは**スレッジハンマー**——スキューブで最も重要なアルゴリズムです：

**スレッジハンマー：R' L R L'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

スレッジハンマーとセットアップムーブを使って、各白い角を下層に配置します。角の挿入例：

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

**コツ：**
- まず白いセンターを下にします。
- 角を1つずつ揃え、すでに配置されたピースを動かさないよう注意してください。
- このステップ後、下面は完全に白く、各隣接面には対応する角の色があるべきです。

---

## ステップ2：対面のセンターを配置する

**上のセンターと4つの側面のセンター**をすべて正しい位置に調整します。下面は揃ったままです。

スレッジハンマーが再び重要なアルゴリズムです。上のセンターと側面のセンターを入れ替えます：

**ケース1：2つのセンターを入れ替える** — スレッジハンマーで上と前のセンターを入れ替えます。

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

**ケース2：3つのセンターの巡回** — 3つのセンターを回転させる場合、スレッジハンマーを2回適用します（ダブルスレッジハンマー）：

**ダブルスレッジハンマー：R' L R L' R' L R L'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R L R' L' R"
  alg="R' L R L' R' L R L'"
/>

**コツ：**
- アルゴリズムを適用する前に、パズルを回転させて正しいセンターを前と上の位置にセットアップしてください。
- 2つのセンターだけ入れ替える場合、スレッジハンマー1回で十分です。3つのセンターを巡回させる場合、ダブルスレッジハンマーを使ってください。

---

## ステップ3：最終層の角の向きを揃える

最後のステップ！**上層の4つの角**を回転させて色が正しい方向を向くようにし、解法を完成させます。

**ケース1：4つの角すべてが時計回りの回転が必要** — このアルゴリズムを適用します：

**アルゴリズム：R L' R' L R L' R' L**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L' R L R' L' R L R'"
  alg="R L' R' L R L' R' L"
/>

**ケース2：一部の角の調整が必要** — セットアップムーブとスレッジハンマーで個々の角の向きを揃えます：

**アルゴリズム：R L' R' L**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L' R L R'"
  alg="R L' R' L"
/>

**コツ：**
- 一部の角だけ回転が必要な場合、セットアップムーブ（UまたはU'）で未解決の角を位置に持ってきて、アルゴリズムを適用し、セットアップムーブを戻してください。
- 残りの角についてプロセスを繰り返す必要があるかもしれません。

---

**おめでとうございます！** スキューブが完成しました！スキューブは必要なアルゴリズムが少ないため、最も達成感のあるパズルの1つです。練習すれば10秒以内で解けるようになります。スレッジハンマーの練習を続けましょう——このパズルをマスターする鍵です。
