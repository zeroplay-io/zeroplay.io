# F2L：最初の2層

F2L（最初の2層）はCFOP法の中で**最も重要なステップ**です。初心者向け解法のように1層目のコーナーと2層目のエッジを別々の2ステップで解くのではなく、F2Lではコーナーとそれに合うエッジをペアにして、正しいスロットに一緒に挿入します。これにより手数が大幅に削減され、最もスピードアップに効果的な部分です。

底面の2層の四隅にある**4つのスロット**を埋める必要があります。各スロットはコーナーピース1つとエッジピース1つで構成されます。

## コンセプト

すべてのF2Lケースは2つのピースに関係します：
- 底層に属する**コーナーピース**（白色/底面色のステッカーがある）
- 中層に属する**エッジピース**（2つの側面色をつなぐ）

目標はこの2つのピースを上層で**ペアにし**、正しいスロットにスムーズに**挿入**することです。

## 回転記号の確認

- **R** = 右面 &nbsp;&nbsp; **L** = 左面 &nbsp;&nbsp; **U** = 上面 &nbsp;&nbsp; **D** = 下面 &nbsp;&nbsp; **F** = 前面 &nbsp;&nbsp; **B** = 後面
- 単独の文字 = **時計回り90°** &nbsp;&nbsp; **'** = **反時計回り** &nbsp;&nbsp; **2** = **180°**

キューブを**白を下**にして持ちます。挿入先のスロットは常に**前面右スロット**（F面とR面の間）です。

---

## 直感的F2L

手順を暗記する前に、F2Lの**ロジック**を学びましょう。すべてのケースは同じパターンに従います：

1. 一緒に属するコーナーとエッジのペアを**見つける**
2. 間違ってくっついている場合は**分離する**
3. 一方のピースを**配置し**、もう一方を**持ってきて**合わせる
4. 上層で白が外側または下を向くように**ペアにする**
5. 接続されたペアをスロットに**挿入する**

**重要な原則：** R操作とU操作でコーナーを前面右スロットから取り出したり入れたりし、U操作でエッジを配置します。すでに解決済みのスロットを崩さないでください。

## 基本ケース：両方のピースが上層にある

最もシンプルで一般的なケースです。コーナーとエッジがともにU層にあり、ターゲットスロットが空いています。

コーナーの白がR面を向いており、エッジはすでにU1回でペアになる位置にあります。最も基本的なF2L挿入です。

## より速いF2Lのためのコツ

### 1. まず直感的に学ぶ

41のアルゴリズムケースを急いで暗記しないでください。各操作が**なぜ**機能するのかを理解する時間をかけましょう。R U R'がコーナーを取り出して戻す操作だと分かれば、ほとんどのケースを自分で考え出せます。アルゴリズムは、すでに理解した手順を最適化するために後から使います。

### 2. 先読み（ルックアヘッド）を鍛える

**先読み**とは、現在のペアを挿入しながら次のペアを追跡する能力です。これが20秒キューバーと15秒キューバーを分ける最大の要因です。

練習のヒント：
- 回転を遅くして、挿入中に次のペアを探すことを強制する
- 挿入中に目を閉じてF2Lを練習する（手順は分かっているので、先を見る）
- ペア間のゼロポーズに集中したタイム計測なしの練習

### 3. 持ち替えを減らす

キューブ全体を回転させるたびに時間を失います。最高のF2Lキューバーは、F2Lフェーズ全体で1〜2回以上回転することはめったにありません。

- RとU操作を使って**後ろのスロット**への挿入を学ぶ（例：R U' R'で右後ろスロットに挿入）
- 反対側のスロットには**L、F、B**操作の使用を練習する
- ワイドムーブ（r, l）で回転と挿入を同時に行う

### 4. ケースを素早く識別する

どのケースかを瞬時に識別するトレーニング：
- **コーナーの白はどこ？**（上、前、右）
- **エッジはコーナーに対してどこ？**（隣接、反対、スロット内）
- **すでにペアになっている？**（正しく接続、間違って接続、離れている）

### 5. 効率的な順序で解く

4ペアを決まった順序で解く必要はありません。4ペアすべてを見て**最も簡単な**ものから始めましょう — 最少手数のものです。経験者はインスペクション中にクロスと一緒に最初のペアを計画します。

### 6. 空きスロットの活用

スロットが空いているとき、一時的な保管場所として使えます。例えば、別のペアに取り組んでいる間、エッジを空きスロットに入れておく（R U R'）ことができます。初心者はこの強力なテクニックを見落としがちです。

---

## 練習メニュー

1. **キューブをスクランブルしてクロスを解く**
2. **4つのコーナーとエッジのペアをすべて見つける** — 各コーナーとそのマッチングエッジの位置を特定
3. **一つずつペアを解く** — 簡単なケースから始め、難しいものは後に残す
4. **F2Lフェーズのみを計測する** — タイマーを使用。目標：4ペアで15秒以下
5. **振り返り** — 毎回のソルブ後に問う：どのペアをもっと少ない手数でできたか？

継続的な練習で、F2Lは第二の本能となり、全体のソルブタイムに劇的な改善が見られるでしょう。
