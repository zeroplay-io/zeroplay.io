# スキューブ上級テクニック

スキューブの解法を初心者レベルから競技レベルへ引き上げましょう。このガイドでは、初心者法をアップグレードし残りのセンターをワンステップで解く **Sarah 中級法**、そしてトップスキューブソルバーが使う上級手順とスピードソルビング戦略を解説します。

## 記号の確認

スキューブは面ではなく**コーナー**を中心に回転します：

- **R** = 右前コーナー時計回り &nbsp;&nbsp; **R'** = 反時計回り
- **L** = 左前コーナー時計回り &nbsp;&nbsp; **L'** = 反時計回り
- **U** = 右前上コーナー時計回り &nbsp;&nbsp; **U'** = 反時計回り
- **B** = 右奥コーナー時計回り &nbsp;&nbsp; **B'** = 反時計回り

すべてのステップで、スキューブの**白センターを下**にして持ちます。

---

## Sarah 中級法

Sarah 初心者法はスキューブを3ステップで解きます：最初の面、次にセンター、最後に最終層コーナー。中級アップグレードでは**センターとコーナーのステップを統合**し、最初の面が完成した後に1つの手順で残り5つのセンターすべてを解きます。

### 概要

| ステップ | 初心者法 | 中級法 |
|------|----------|-------------|
| 1 | 一面を完成 | 一面を完成（同じ） |
| 2 | センターを配置 | **5つのセンターをワンステップで解く** |
| 3 | 最終層コーナーの向きを合わせる | 最終層コーナーの向きを合わせる（同じ） |

重要な改善点はステップ2です：Sledgehammer を繰り返してセンターを1つずつ配置する代わりに、すべてのセンターを同時に解く手順セットを学びます。

### ステップ1：一面を完成させる

底面に完全な**白い面**を作ります — 白センター周りの4つのコーナーが正しい位置に白を下向きにして配置され、隣接する側面の色が一致している必要があります。このステップは初心者法と同じです。

**Sledgehammer: R' L R L'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

Sledgehammer をセットアップムーブと組み合わせて各白コーナーを配置します。練習すればこのステップは素早く直感的になります。

### ステップ2：5つのセンターを一度に解く（One-Look）

最初の面が完成したら、**残り5つのセンター**（上面 + 4側面）を観察し、どのケースかを判断します。1つの手順ですべてを一度に解きます。

正しい位置にあるセンターの数に応じて、いくつかのケースがあります：

---

#### ケース1：隣接4サイクル

4つのセンターがサイクルする必要があります。上面のセンターは正しいですが、4つの側面センターが回転する必要があります。

**手順：R' L R L' R' L R L'**（ダブル Sledgehammer）

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R L R' L' R"
  alg="R' L R L' R' L R L'"
/>

#### ケース2：対面2交換

2つの対面にある側面センターを交換する必要があります。

**手順：R L' R' L R L' R' L**（逆ダブル Sledgehammer）

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L' R L R' L' R L R'"
  alg="R L' R' L R L' R' L"
/>

#### ケース3：隣接2交換

2つの隣接する側面センターを交換する必要があります。交換する2つのセンターを前面と右面にセットアップします。

**手順：R' L R L' U R' L R L' U'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="U L R' L' R U' L R' L' R"
  alg="R' L R L' U R' L R L' U'"
/>

#### ケース4：上面-側面交換 + サイクル

上面のセンターを側面のセンターと交換し、残りの側面センターがサイクルします。

**手順：U R' L R L' U'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="U L R' L' R U'"
  alg="U R' L R L' U'"
/>

#### ケース5：トリプルサイクル

3つのセンターがサイクルします（上面は固定、3つの側面が回転）。固定の側面センターを背面にセットアップします。

**手順：R' L R L'**（シングル Sledgehammer — 正しいセットアップで）

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

**ヒント：**
- 上面のセンターを最初に確認し、次に4つの側面をスキャンして、センターケースを素早く識別する方法を学びましょう。
- これらの多くは Sledgehammer のバリエーションです — Sledgehammer を十分に理解していれば、これらの手順は自然に感じられます。
- スキューブをスクランブルし、最初の面を解き、実行する前にセンターケースを判断する練習をしましょう。

### ステップ3：最終層コーナーの向きを合わせる

すべてのセンターを解いた後、最終層のコーナーはまだねじる必要があるかもしれません。このステップは初心者法と同じです。

#### 4つのコーナーすべてが時計回りのねじれが必要

**手順：R L' R' L R L' R' L**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L' R L R' L' R L R'"
  alg="R L' R' L R L' R' L"
/>

#### 4つのコーナーすべてが反時計回りのねじれが必要

**手順：L' R L R' L' R L R'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="R L' R' L R L' R' L"
  alg="L' R L R' L' R L R'"
/>

#### 2つのコーナーがねじれ必要（隣接）

未解決の2つのコーナーを前面にセットアップします。U で位置を合わせ、手順を実行し、U を元に戻します。

**手順：U R L' R' L R L' R' L U'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="U L' R L R' L' R L R' U'"
  alg="U R L' R' L R L' R' L U'"
/>

#### 2つのコーナーがねじれ必要（対角）

**手順：R L' R' L U R L' R' L U'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="U L' R L R' U' L' R L R'"
  alg="R L' R' L U R L' R' L U'"

## 上級手順

中級法を超えて、あまり一般的でないケースをより効率的に処理する追加手順を紹介します。

### Z-Perm 等価手順

2組のコーナーが交換する必要がある場合（Z字パターン）：

**手順：R' L R L' U R' L R L' U R' L R L'**

上面が Pi パターンを示す — 2つの隣接コーナーが正しい向き、2つはそうでない：

## One-Look ソルビングの概念

競技スキューブソルビングの究極の目標は、最初の面を解きながら最終層を予測することです。

### 最終層の予測

ステップ1で白い面を作る際、**上層のセンターとコーナー**に注目してください。練習を重ねれば、ステップ1を終える前にステップ2とステップ3のケースを識別できるようになります。

- **上面のセンターを観察する** — 最後の白コーナーを挿入する際、正しい位置に移動しますか？
- **側面センターを追跡する** — どの側面センターが既に正しいか注目しましょう。
- **完全なソルブを計画する** — 上級ソルバーはインスペクション中にソルブ全体を計画できます。

### 最終層への影響

最後の白コーナーを挿入する方法が複数ある場合、**より簡単なステップ2のケース**になる挿入を選びましょう：

- 2つの挿入がどちらもコーナーを解く場合、より多くのセンターが正しい位置になる方を選びます。
- ステップ1が少し長くなっても、ステップ2やステップ3がスキップになることがあります。

## スピードソルビングのコツ

### 持ち方と回し方

- スキューブは**コーナーに指先を置いて**持ちます — 面ではなくコーナーを回します。
- 各手の**人差し指と親指**で対角のコーナーを掴みます。
- 滑らかで制御されたコーナーターンを練習しましょう — スキューブは速さよりも精度が報われます。
- **持ち替えを最小限に** — 持ち方を変えずに Sledgehammer 全体を実行できるよう練習しましょう。

### カラーニュートラル

スキューブのカラーニュートラルとは、白だけでなく**どの面からでも**最初の面として始められることを意味します。これは非常に強力です：

- カラーニュートラルなら、**6つの選択可能な最初の面**があります。
- 最も完成に近い面を選びましょう — 時には2〜3個のコーナーが既に揃っています。
- カラーニュートラルは4秒以下のソルブにはほぼ必須です。

**練習方法：**
1. まず2つの対面色の間でニュートラルになりましょう（例：白と黄色）。
2. 徐々に開始色を増やし、どの面からでも始められるようにします。
3. インスペクション中に6面すべてを素早くスキャンし、最も簡単な面を選びます。

### インスペクション計画

スキューブは可視ピースがわずか14個（6センター + 8コーナー）なので、フルソルブの計画が現実的です：

- インスペクション中に**最初の面を常に完全に計画**しましょう。
- 解き始める前に**センターケースを特定**しましょう。
- **上級**：ステップ2まで計画する — パズルに触れる前にどの手順を使うか把握しておきましょう。

### 練習ルーティン

1. **最初の面ドリル** — スクランブルして最初の面だけを解く、目標は2秒以内。
2. **センター認識** — 最初の面を解いた後、実行せずにできるだけ早くセンターケースを特定する。
3. **フルソルブ** — 12回のタイムドソルブのセット、5回平均と12回平均を記録。
4. **手順ドリル** — 各センターとコーナーの手順を20回連続で練習し、筋肉記憶を作る。
5. **カラーニュートラル練習** — 毎回異なる色からソルブを始めることを自分に課す。

---

**Sarah 中級法と継続的な練習で、5秒以下のスキューブ平均は手の届く範囲にあります。**スキューブはスピードキュービングで最も計画しやすいパズルの1つです — ピースが少ないため、インスペクション中により多くを予測できます。認識速度と滑らかな回転に集中すれば、急速な上達が見られるでしょう。
