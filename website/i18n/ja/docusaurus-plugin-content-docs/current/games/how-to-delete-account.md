# アカウントの削除方法

**ZeroPlay** のゲームからアカウントを削除したい場合は、以下の手順に従ってください:

---

### ステップ 1: プロフィールにアクセス

下部のナビゲーションバーの *マイページ* タブをタップして、プロフィールページにアクセスします。

![プロフィール](https://dl.zeroplay.cc/metadata/how-to-delete-account/1.jpg)

### ステップ 2: 削除オプションを見つける

<span className="danger-text">危険ゾーン</span> セクションまでスクロールし、<span className="danger-text">アカウント削除</span> ボタンをタップします。

![アカウント削除](https://dl.zeroplay.cc/metadata/how-to-delete-account/2.jpg)

### ステップ 3: 削除を確認

警告を確認し、*続行* をタップしてアカウント削除を続行します。

![削除の確認](https://dl.zeroplay.cc/metadata/how-to-delete-account/3.jpg)

### ステップ 4: 最終カウントダウン

削除プロセスは10秒のカウントダウンで開始されます。この間、*削除をキャンセル* をタップしてキャンセルすることができます。

![アカウント削除中](https://dl.zeroplay.cc/metadata/how-to-delete-account/4.jpg)

---

### ⚠️ 重要な注意事項

- **永久的な操作** — すべてのゲームデータが永久に削除されます
- **復元不可** — カウントダウン終了後、この操作は元に戻せません
