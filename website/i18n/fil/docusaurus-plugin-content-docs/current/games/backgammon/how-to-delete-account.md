# Paano Tanggalin ang Account

Kung gusto mong tanggalin ang iyong account mula sa **Backgammon - Fair Board Games**, sundin ang mga hakbang na ito:

---

### Hakbang 1: Buksan ang Settings

Ilunsad ang laro at i-tap ang *Settings* button sa kanang sulok sa itaas.

![Settings](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/1.jpg#landscape)

### Hakbang 2: Hanapin ang Pagpipilian sa Pagtanggal

Mag-scroll sa ibaba ng Settings window at i-tap ang button na <span className="danger-text">Delete Account</span>.

![Delete Account](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/2.jpg#landscape)

### Hakbang 3: I-verify ang Iyong Aksyon

I-type ang confirmation code na ipinapakita sa screen, pagkatapos ay i-tap ang *Delete* button upang magpatuloy.

![Confirm Deletion](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/3.jpg#landscape)

### Hakbang 4: Huling Countdown

Ang proseso ng pagtanggal ay nagsisimula sa 10-segundo countdown. Maaari mo pa ring kanselahin sa pamamagitan ng pag-tap sa *Cancel Deletion* sa panahong ito.

![Account Deleted](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/4.jpg#landscape)

---

### ⚠️ Mahahalagang Paalala

- **Permanenteng Aksyon** — Ang lahat ng iyong game data ay permanenteng matatanggal
- **Walang Pagbawi** — Ang aksyong ito ay hindi maaaring bawiin pagkatapos ng countdown
