# Paano Tanggalin ang Account

Kung gusto mong tanggalin ang iyong account mula sa anumang laro ng **ZeroPlay**, sundin ang mga hakbang na ito:

---

### Hakbang 1: I-access ang Iyong Profile

I-tap ang *Me* tab sa ibabang navigation bar upang ma-access ang iyong profile page.

![Profile](https://dl.zeroplay.cc/metadata/how-to-delete-account/1.jpg)

### Hakbang 2: Hanapin ang Pagpipilian sa Pagtanggal

Mag-scroll pababa sa seksyon ng <span className="danger-text">DANGER ZONE</span> at i-tap ang button na <span className="danger-text">Delete Account</span>.

![Delete Account](https://dl.zeroplay.cc/metadata/how-to-delete-account/2.jpg)

### Hakbang 3: Kumpirmahin ang Pagtanggal

Suriin ang babala at i-tap ang *Continue* upang magpatuloy sa pagtanggal ng account.

![Confirm Deletion](https://dl.zeroplay.cc/metadata/how-to-delete-account/3.jpg)

### Hakbang 4: Huling Countdown

Ang proseso ng pagtanggal ay nagsisimula sa 10-segundo countdown. Maaari mo pa ring kanselahin sa pamamagitan ng pag-tap sa *Cancel Deletion* sa panahong ito.

![Account Deletion in Progress](https://dl.zeroplay.cc/metadata/how-to-delete-account/4.jpg)

---

### ⚠️ Mahahalagang Paalala

- **Permanenteng Aksyon** — Ang lahat ng iyong game data ay permanenteng matatanggal
- **Walang Pagbawi** — Ang aksyong ito ay hindi maaaring bawiin pagkatapos ng countdown
