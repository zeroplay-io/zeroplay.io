# Как собрать Скьюб

Научитесь собирать Скьюб с помощью **метода Сары для начинающих** — простого и эффективного подхода. Скьюб — это головоломка с вращением по углам, где каждый ход вращает угол и три прилегающие грани. Мы соберём его всего за 3 шага, используя один ключевой алгоритм: Кувалду (Sledgehammer).

## Обозначения

В отличие от стандартного кубика Рубика, Скьюб вращается вокруг **углов**, а не граней. Каждый ход вращает один угол и три соседних центра:

- **R** = Правый передний угол по часовой стрелке &nbsp;&nbsp; **R'** = Правый передний угол против часовой стрелки
- **L** = Левый передний угол по часовой стрелке &nbsp;&nbsp; **L'** = Левый передний угол против часовой стрелки
- **U** = Верхний правый передний угол по часовой стрелке &nbsp;&nbsp; **U'** = Верхний правый передний угол против часовой стрелки
- **B** = Задний правый угол по часовой стрелке &nbsp;&nbsp; **B'** = Задний правый угол против часовой стрелки

Буква без символа означает вращение **по часовой стрелке**. Добавление **'** означает **против часовой стрелки**.

Держите Скьюб **белым центром вниз** на всех этапах.

---

## Шаг 1: Собрать одну грань

Соберите полную **белую грань** внизу. Это означает, что все 4 угла вокруг белого центра должны быть на своих местах белой стороной вниз, а соседние боковые цвета должны совпадать.

Этот шаг полуинтуитивный. Ключевой инструмент — **Кувалда (Sledgehammer)** — самый важный алгоритм для Скьюба:

**Кувалда: R' L R L'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

Используйте Кувалду и подготовительные ходы, чтобы поставить каждый белый угол в нижний слой. Вот пример вставки угла:

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

**Советы:**
- Начните с размещения белого центра внизу.
- Собирайте углы по одному, стараясь не нарушить уже установленные элементы.
- После этого шага нижняя грань должна быть полностью белой, а каждая соседняя сторона должна иметь соответствующий цвет угла.

---

## Шаг 2: Расположить противоположные центры

Настройте **верхний центр и 4 боковых центра** так, чтобы все они были в правильных позициях. Нижняя грань остаётся собранной.

Кувалда снова является ключевым алгоритмом. Она меняет местами верхний центр с боковым:

**Случай 1: Поменять 2 центра** — Используйте Кувалду, чтобы поменять верхний и передний центры.

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R"
  alg="R' L R L'"
/>

**Случай 2: 3-цикл центров** — Когда три центра нужно циклически переместить, примените Кувалду дважды (Двойная Кувалда):

**Двойная Кувалда: R' L R L' R' L R L'**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L R' L' R L R' L' R"
  alg="R' L R L' R' L R L'"
/>

**Советы:**
- Поверните головоломку, чтобы правильные центры оказались в передней и верхней позициях, прежде чем применять алгоритм.
- Если нужно поменять только 2 центра, одной Кувалды достаточно. Если 3 центра нужно циклически переместить, используйте Двойную Кувалду.

---

## Шаг 3: Ориентация углов последнего слоя

Последний шаг! Поверните **4 угла верхнего слоя** так, чтобы их цвета были направлены правильно, завершая сборку.

**Случай 1: Все 4 угла требуют поворота по часовой стрелке** — Примените этот алгоритм:

**Алгоритм: R L' R' L R L' R' L**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L' R L R' L' R L R'"
  alg="R L' R' L R L' R' L"
/>

**Случай 2: Некоторые углы требуют корректировки** — Используйте Кувалду с подготовительными ходами для ориентации отдельных углов:

**Алгоритм: R L' R' L**

<TwistyPlayer
  puzzle="skewb"
  setupAlg="L' R L R'"
  alg="R L' R' L"
/>

**Советы:**
- Если только некоторые углы нужно повернуть, используйте подготовительные ходы (U или U'), чтобы привести несобранные углы в позицию, примените алгоритм, затем отмените подготовительный ход.
- Возможно, потребуется повторить процесс для оставшихся углов.

---

**Поздравляем!** Вы собрали Скьюб! Скьюб — одна из самых приятных головоломок, потому что для неё нужно очень мало алгоритмов. С практикой вы будете собирать его менее чем за 10 секунд. Продолжайте практиковать Кувалду — это ключ к мастерству в этой головоломке.
