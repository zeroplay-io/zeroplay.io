# Как собрать кубик Рубика 4x4

Научитесь собирать кубик Рубика 4x4 (также называемый **Rubik's Revenge**) с помощью **метода редукции** — самого популярного подхода для больших кубиков. Идея проста: сначала привести 4x4 к виду 3x3, а затем собрать как 3x3. Мы рассмотрим 4 шага плюс специальные случаи паритета.

## Обозначения

В 4x4 используются все стандартные обозначения 3x3 плюс дополнительные ходы для внутренних слоёв:

**Стандартные ходы** (только внешний слой):
- **R** = Правый &nbsp;&nbsp; **L** = Левый &nbsp;&nbsp; **U** = Верхний &nbsp;&nbsp; **D** = Нижний &nbsp;&nbsp; **F** = Передний &nbsp;&nbsp; **B** = Задний

**Широкие ходы** (два внешних слоя вместе):
- **Rw** = Правый широкий (два правых слоя) &nbsp;&nbsp; **Lw** = Левый широкий
- **Uw** = Верхний широкий (два верхних слоя) &nbsp;&nbsp; **Dw** = Нижний широкий
- **Fw** = Передний широкий &nbsp;&nbsp; **Bw** = Задний широкий

Буква без символов означает **90° по часовой стрелке**. С **'** — **против часовой стрелки**. С **2** — **180°**.

Например: **Rw** = два правых слоя по часовой стрелке, **Uw'** = два верхних слоя против часовой стрелки, **Rw2** = два правых слоя на 180°.

Держите кубик **белым центром вниз** и **жёлтым центром вверх** на всех этапах.

---

## Шаг 1: Собрать центры

Соберите **центральные блоки** на каждой грани. Каждая грань 4x4 имеет 4 центральных элемента (блок 2x2), которые нужно собрать. В отличие от 3x3, центры 4x4 могут перемещаться — поэтому их нужно собирать первыми.

**Рекомендуемый порядок:** Начните с **белого** (низ), затем **жёлтый** (противоположная сторона), потом оставшиеся 4 грани.

Этот шаг в основном **интуитивный**. Ключевая техника — использовать широкие ходы для объединения центральных элементов, защищая уже собранные грани поворотами внешнего слоя.

**Базовая вставка центра** — Используйте широкий ход, чтобы поднять центральный элемент наверх, поверните верхний слой для выравнивания, затем верните широкий ход:

<TwistyPlayer
  puzzle="4x4x4"
  setupAlg="Rw U Rw'"
  alg="Rw U' Rw'"
/>

**Обмен центров без разрушения собранных граней** — Когда нужно вставить центральный элемент, но при этом защитить другие собранные центры:

<TwistyPlayer
  puzzle="4x4x4"
  setupAlg="Rw U Rw' U Rw U2 Rw'"
  alg="Rw U2' Rw' U' Rw U' Rw'"
/>

**Советы:**
- Всегда сначала собирайте противоположные грани (белый/жёлтый), потом остальные 4 грани.
- При вставке центрального элемента используйте широкий ход (как Rw), чтобы поднять его наверх, скорректируйте слой U, затем отмените широкий ход.
- Будьте осторожны, чтобы не разрушить собранные центры при работе над новыми.

---

## Шаг 2: Спаривание рёбер

Соедините каждую пару **рёберных элементов**, чтобы 4x4 стал эквивалентен 3x3. Каждое ребро 4x4 состоит из 2 элементов, которые нужно спарить. Всего 12 рёбер для спаривания.

Основная техника — **Slice-Flip-Slice**: используйте широкий ход (Uw) для выравнивания рёбер, выполните ход внешнего слоя для спаривания, затем отмените широкий ход.

**Базовое спаривание рёбер** — Поместите один рёберный элемент в переднюю правую позицию, а его пару наверх. Используйте Uw для выравнивания:

<TwistyPlayer
  puzzle="4x4x4"
  setupAlg="Uw R U R' Uw'"
  alg="Uw R U' R' Uw'"
/>

**Спаривание с переворотом** — Когда пару рёбер нужно перевернуть при спаривании:

<TwistyPlayer
  puzzle="4x4x4"
  setupAlg="Uw R U' R' Uw'"
  alg="Uw R U R' Uw'"
/>

**Спаривание последних рёбер** — Для последних 3 рёбер нельзя свободно использовать Uw, не разрушая уже спаренные рёбра. Используйте эту технику:

<TwistyPlayer
  puzzle="4x4x4"
  setupAlg="Dw F' R U' F R' Dw'"
  alg="Dw R F' U R' F Dw'"
/>

**Советы:**
- Первые 8-9 рёбер можно свободно спаривать техникой Slice-Flip-Slice.
- Для последних 3 рёбер используйте ходы на основе Dw или технику переворота, чтобы не разрушать уже спаренные рёбра.
- Если пара рёбер на правильном месте, но перевёрнута, нужно извлечь её и спарить заново.

---

## Шаг 3: Собрать как 3x3

После сборки всех центров и спаривания всех рёбер 4x4 теперь эквивалентен 3x3! Используйте любой **метод сборки 3x3** (например, послойный метод) для завершения.

**Важно:** На этом этапе используйте только ходы внешнего слоя (R, U, F, L, D, B). **Не** используйте широкие ходы (Rw, Uw и т.д.), так как они разрушат спаренные рёбра и собранные центры.

<TwistyPlayer
  puzzle="4x4x4"
  alg="R U R' U'"
/>

Обратитесь к нашему [руководству по 3x3](./index.mdx) для полного послойного метода. Применяются те же 7 шагов — белый крест, белые углы, второй слой, жёлтый крест, жёлтая грань, расстановка углов и расстановка рёбер.

---

## Шаг 4: Алгоритмы паритета

У 4x4 есть две особые ситуации, которые **никогда не встречаются на 3x3**. Они называются случаями **паритета** и каждый возникает примерно в 50% случаев. Если вы столкнётесь с ним на этапе 3x3, примените соответствующий алгоритм и продолжайте сборку.

### OLL-паритет (переворот одного ребра)

**Как выглядит:** На последнем слое одно ребро перевёрнуто — невозможная ситуация на настоящем 3x3. Жёлтый крест не может быть завершён обычным способом.

**Алгоритм: Rw U2 x Rw U2 Rw U2 Rw' U2 Lw U2 Rw' U2 Rw U2 Rw' U2 Rw'**

<TwistyPlayer
  puzzle="4x4x4"
  setupAlg="Rw U2 Rw' U2 Rw U2' Lw' U2 Rw U2' Rw' U2 Rw U2' Rw' x' U2' Rw'"
  alg="Rw U2 x Rw U2 Rw U2 Rw' U2 Lw U2 Rw' U2 Rw U2 Rw' U2 Rw'"
/>

Этот алгоритм длинный, но его нужно применить только один раз. После выполнения ребро будет правильно ориентировано, и вы сможете продолжить обычные шаги последнего слоя 3x3.

### PLL-паритет (обмен двух рёбер)

**Как выглядит:** В самом конце нужно поменять местами ровно два ребра — опять же, невозможно на 3x3.

**Алгоритм: 2R2 U2 2R2 Uw2 2R2 Uw2**

<TwistyPlayer
  puzzle="4x4x4"
  setupAlg="Uw2 2R2 Uw2 2R2 U2 2R2"
  alg="2R2 U2 2R2 Uw2 2R2 Uw2"
/>

Этот короткий алгоритм меняет местами два противоположных ребра в последнем слое. После применения кубик должен быть полностью собран.

**Советы:**
- Не паникуйте при встрече с паритетом — это совершенно нормально для кубиков 4x4.
- OLL-паритет появляется на этапе жёлтого креста. PLL-паритет появляется в самом конце.
- Вы можете встретить оба, один или ни одного случая паритета за одну сборку.

---

**Поздравляем!** Вы собрали кубик Рубика 4x4! Метод редукции хорошо масштабируется на ещё бо́льшие кубики (5x5, 6x6 и т.д.) — основные концепции сборки центров, спаривания рёбер и сборки как 3x3 остаются теми же. С практикой вы сможете собирать 4x4 менее чем за 3 минуты.
