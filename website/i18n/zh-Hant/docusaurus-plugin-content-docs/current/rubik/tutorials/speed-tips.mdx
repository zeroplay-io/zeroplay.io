# 速解技巧

提高魔方速度不僅仅是转得更快。最大的進步来自于提前規劃、減少停頓和有目的地練習。本指南涵盖将休闲玩家与競技速解選手区分开来的核心技能。

---

## 十字規劃

十字是 CFOP 還原的第一步，也是唯一可以在**觀察時間内完全規劃**的部分。掌握十字規劃能让你在每次還原中都獲得先机。

### 充分利用15秒觀察時間

在比賽中，你在啟動計時器前有15秒的觀察時間。善用每一秒：
- 前5秒：确定4条十字棱块的位置。
- 中间5秒：規劃完整的十字解法（理想情況下6-8步或更少）。
- 最后5秒：在脑中排练步驟并規劃第一组 F2L。

### 在底面做十字

始终在**底面**（D层）做十字。初學者常在顶面做十字然後翻转魔方，浪費1-2秒并失去方向感。從一開始就在底面做十字，可以在完成十字的同時立即開始尋找 F2L 配對。

**示例：6步十字解法**

从此打亂開始，白色十字仅需6步即可完成：

<TwistyPlayer
  puzzle="3x3x3"
  alg="D R F' D' R' D"
  setupAlg="D' R D F R' D'"
  stickering="cross"
/>

**示例：高效5步十字**

通過良好的規劃，许多十字可以在5步或更少内完成：

<TwistyPlayer
  puzzle="3x3x3"
  alg="R D' F D2 R'"
  setupAlg="R D2 F' D R'"
  stickering="cross"
/>

### 閉眼练十字

這是最好的十字練習方法：
1. 打亂魔方。
2. 在觀察時間内規劃完整的十字解法。
3. 闭上眼睛，憑記憶執行十字。
4. 睜開眼睛驗證。

如果十字正確，說明你的規劃很扎實。如果不正確，找出你的心算在哪裡出了問題。

### 色彩中性

大多数初學者只学白色十字。**色彩中性**的選手可以从6种顏色中任选一种開始，選擇每次打亂中最簡單的十字。这通常能在十字上節省2-3步，平均每次還原節省1-2秒。

要培養色彩中性：
1. 先增加一种顏色（例如黄色，白色的對面）。
2. 每次打亂时，規劃白色和黄色两个十字，選擇步数更少的那個。
3. 逐步增加更多顏色，直到你能熟练使用全部六種。

## 預判（Lookahead）

預判是在**解決當前部分的同時追蹤下一个部件**的能力。這是突破速度瓶颈的最重要技能，也是 sub-15 選手和 sub-10 選手之間的分水岭。

### 預判的真正含義

在 F2L 階段，你每次解決一个角棱配對。没有預判时，流程是：解決配對 - 停頓 - 尋找下一组配對 - 解決配對 - 停頓。这些停頓每次還原累计可达数秒。

有了預判，当你的雙手在執行當前配對的已知公式时，你的眼睛已经在掃描下一组配對的部件。目標是配對之間**零停頓**。

### 如何練習預判

**慢速還原**是最有效的訓練方法：

1. 将節拍器設置为每秒1-2次轉動（60-120 BPM）。
2. 以這個恆定速度還原 F2L，每个操作对准節拍。
3. 規則：**你可以转得慢，但絕不能停下来。**
4. 如果你必須停下来尋找下一组配對，說明你转太快了。繼續放慢。

經過幾週的練習，你会在保持連續流動的同時自然加速。

### 盲拧 F2L 配對練習

1. 打亂魔方。
2. 還原十字。
3. 找到一组 F2L 配對并在脑中規劃插入。
4. 闭上眼睛執行插入。
5. 睜開眼睛，重复下一组配對。

这迫使你的大腦在執行前完全規劃每次插入，增強真正預判所需的心算追蹤能力。

## 高效 F2L

F2L（前两层）階段有最大的優化空間。即使 F2L 效率的微小提升也会在數百次還原中累積。

### 避免不必要的整體旋轉

整體旋轉（y, y'）耗費時間并打斷預判。典型初學者的 F2L 有6-8次旋轉，高級選手平均只有1-2次。

**示例：不旋轉插入后右槽位**

不要用 y' 旋轉后插入前右槽位，而是使用后槽位插入：

<TwistyPlayer
  puzzle="3x3x3"
  alg="R U' R'"
  setupAlg="R U R'"
/>

**对比需要旋轉的相同配對：**

<TwistyPlayer
  puzzle="3x3x3"
  alg="y' R U' R'"
  setupAlg="R U R' y"
/>

第一种方式節省约0.5秒——旋轉本身加上换手。

### 优先后槽位

将 F2L 配對插入**后右**和**后左**槽位通常更高效，因为：
- R 和 L 操作比 F 操作更快。
- 后槽位插入通常需要更少步驟。
- 当未解決的部件在你面前可見时，你能保持更好的預判。

### 影響下一组配對

在插入當前 F2L 配對时，你有時可以免費設置下一组配對。例如，如果下一组配對的角塊在 U 层，選擇當前配對特定的插入角度可能将该角塊移到有利位置。

這是一项進階技能，但即使僅僅意識到它就能幫助你做出更好的決策。

### 多槽位插入

在某些情況下，你可以同時解決两组 F2L 配對，或在解決一组的同時部分設置另一组。这在實踐中很少見，但值得在機會出現時加以利用。

## OLL/PLL 識別

快速識別 OLL 和 PLL 情況可以消除 F2L 和最后一层之間的停頓。

### 从两个角度識別

練習通過同時觀察**顶面**和**前面**来識別 OLL 情況。这能在不旋轉魔方檢查其他面的情況下給你所有需要的信息。

對於 PLL，檢查**前面和右面**（从你標準解题角度可見的两个面）。通過練習，两面識別会变得即时。

### 预 AUF

AUF（Adjust U Face）是你在開始 OLL 或 PLL 公式之前做的 U/U'/U2 操作，用於正確對齊情況。高級選手在 F2L 的最后一步或識別过程中執行 AUF，与思考時間重疊。

### OLL 預測

在最后一组 F2L 插入时，你通常已知4个顶面角塊方向中的2-3个。通過追蹤你的插入如何影響最后一个角塊，你可以在完成 F2L 之前預測 OLL 情況，并以零停頓開始執行 OLL 公式。

## 練習计划

有目的的練習勝過無腦刷時間。合理安排你的練習以獲得最大提升。

### 每日練習计划（45-60分鐘）

**十字訓練（10分鐘）**
- 打亂、觀察时規劃十字、只還原十字。
- 目標：穩定在2秒以内，閉眼規劃。
- 追蹤你的平均步数。目標6步或更少。

**F2L 慢速還原（15分鐘）**
- 使用節拍器。以恆定转速完成整个 F2L。
- 纯粹專注于預判和連續流動。
- 不要計時。这里速度不是目標。

**公式訓練（10分鐘）**
- 選擇2-3个你还在学习的 OLL 或 PLL 公式。
- 每个重复20-30次直到形成肌肉記憶。
- 然後打亂練習識別 + 執行。

**計時還原（15-20分鐘）**
- 做一組 **ao12**——十二次計時還原，去掉最好和最差的，取剩餘十次的平均值。
- 記錄你的成績。追蹤數週的趨勢，而非單次還原。
- 每次練習后，回顧你最差的那次還原。哪里出了問題？

### 平均值与追蹤

- **Ao5**（5次平均）：快速檢查當前状态。適合熱身。
- **Ao12**（12次平均）：標準競技指標。最可靠的水準指標。
- **Ao100**：長期趨勢追蹤。每周計算以衡量進步。

使用一个能自動追蹤历史記錄的計時器應用。看到你的平均值隨週下降是強大的動力。

### 刻意練習 vs 無腦練習

邊看電視邊做200次還原不会让你变快。刻意練習意味著：
- 每次練習選擇**一个特定技能**来專注（十字、預判、OLL識別）。
- 在每次還原中全神貫注于那個技能。
- 分析錯誤而不是立即重新打亂。

十次專注的還原勝過一百次自動驾驶的還原。

## 比賽技巧

比賽和在家練習是不同的技能。緊張、陌生環境和時間壓力都会影響表現。

### 充分熱身

- 提前到場，在你的項目前做20-30次隨意還原。
- 做公式訓練来喚醒肌肉記憶。
- 熱身时不要試圖刷 PB。只是讓手動起來。

### 還原时保持冷靜

- **專注于流暢還原，而不是快速還原。** 強行追求速度会導致失誤和卡頓。
- 如果你搞砸了一组配對或公式，深呼吸然後繼續。不要恐慌或試圖加速彌補時間。
- 記住：你的比賽平均值会去掉最好和最差的還原，所以一次糟糕的還原不会毁掉你的成績。

### 觀察技巧

- 觀察时将魔方舉到眼睛高度。不要低頭看桌子。
- 用 y 和 x 旋轉高效地查看所有面。
- 完全規劃你的十字。如果時間允許，定位你的第一组 F2L 配對。
- 注意8秒和12秒的警告，合理安排節奏。

### 熟悉 Stackmat 計時器

- 如果你打算參賽，在家用 Stackmat 計時器練習。手的放置和啟停動作与點擊手機螢幕不同。
- 雙手必須平放在墊子上。当你同時抬起雙手时計時器啟動。
- 練習平穩啟動，這樣你就不会在十字的第一步手忙腳亂。

## 硬件

一个好的速解魔方確實有區別。你不需要最貴的魔方来進步，但从普通品牌魔方升級到現代速解魔方是你能做的最大單一提升之一。

### 什麼造就好的速解魔方

- **容錯性**：魔方应该容忍輕微的對齊偏差而不会卡住。現代魔方可以容錯45度或更多。
- **順滑、可控的手感**：不能太快（会過衝）也不能太慢（需要額外力氣）。
- **穩定性**：魔方在快速轉動时应保持形狀，不会飛稜或變形。

### 磁力魔方

如今幾乎所有競技選手都使用**磁力魔方**。每个零件內的小磁鐵在每90度轉動时产生微妙的吸附感，提供：
- 高速时更好的控制
- 更少的過衝和卡頓
- 更舒適的手感

MoYu、QiYi 和 GAN 等品牌的磁力魔方有各種價位。即使是入門級磁力魔方（低於10美元）也比非磁力的好得多。

### 鬆緊和潤滑

- **鬆緊度**控制魔方的鬆緊手感。更松允許更快轉動但降低穩定性。从中等鬆緊度開始，根据你的轉動风格調整。
- **潤滑**減少摩擦并改變魔方手感。矽基潤滑油是標準：
  - **快速潤滑油**用於零件上以提高速度。
  - **慢速/黏性潤滑油**用於核心以提高穩定性。
  - 每幾百次還原或當魔方開始感覺粗糙时重新潤滑。

### 初學者設置建議

1. 購買任何現代磁力3阶（MoYu RS3M、QiYi Tornado 或類似產品）。
2. 第一周保持出廠鬆緊度以適應魔方。
3. 如果魔方感覺乾燥，添加少量矽基潤滑油。
4. 逐步調整鬆緊度——每次鬆四分之一圈，直到找到你喜歡的手感。

一个調整好的魔方消除了硬件作为變量，讓你能完全專注于提高技術。
