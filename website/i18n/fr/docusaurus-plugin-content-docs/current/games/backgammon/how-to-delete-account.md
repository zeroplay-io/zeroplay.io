# Comment Supprimer le Compte

Si vous souhaitez supprimer votre compte de **Backgammon - Fair Board Games**, suivez ces étapes :

---

### Étape 1 : Ouvrez les Paramètres

Lancez le jeu et appuyez sur le bouton *Paramètres* en haut à droite.

![Paramètres](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/1.jpg#landscape)

### Étape 2 : Localisez l'option de suppression

Faites défiler jusqu'au bas de la fenêtre Paramètres et appuyez sur le bouton <span className="danger-text">Supprimer le compte</span>.

![Supprimer le compte](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/2.jpg#landscape)

### Étape 3 : Vérifiez votre action

Saisissez le code de confirmation affiché à l'écran, puis appuyez sur le bouton *Supprimer* pour continuer.

![Confirmer la suppression](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/3.jpg#landscape)

### Étape 4 : Compte à rebours final

Le processus de suppression commence par un compte à rebours de 10 secondes. Vous pouvez encore annuler en appuyant sur *Annuler la suppression* pendant ce temps.

![Compte supprimé](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/4.jpg#landscape)

---

### ⚠️ Remarques importantes

- **Action permanente** — Toutes vos données de jeu seront définitivement supprimées
- **Aucune récupération** — Cette action ne peut pas être annulée après la fin du compte à rebours
