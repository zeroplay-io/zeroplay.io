# Comment Supprimer le Compte

Si vous souhaitez supprimer votre compte de n'importe quel jeu **ZeroPlay**, suivez ces étapes :
<!-- Button labels remain in English to match UI screenshots: Me, DANGER ZONE, Delete Account, Continue, Cancel Deletion. -->

---

### Étape 1 : Accédez à votre Profil

Appuyez sur l'onglet *Me* dans la barre de navigation inférieure pour accéder à votre page de profil.

![Profil](https://dl.zeroplay.cc/metadata/how-to-delete-account/1.jpg)

### Étape 2 : Localisez l'option de suppression

Faites défiler jusqu'à la section <span className="danger-text">DANGER ZONE</span> et appuyez sur le bouton <span className="danger-text">Delete Account</span>.

![Supprimer le compte](https://dl.zeroplay.cc/metadata/how-to-delete-account/2.jpg)

### Étape 3 : Confirmez la suppression

Examinez l'avertissement et appuyez sur *Continue* pour procéder à la suppression du compte.

![Confirmer la suppression](https://dl.zeroplay.cc/metadata/how-to-delete-account/3.jpg)

### Étape 4 : Compte à rebours final

Le processus de suppression commence par un compte à rebours de 10 secondes. Vous pouvez encore annuler en appuyant sur *Cancel Deletion* pendant ce temps.

![Suppression du compte en cours](https://dl.zeroplay.cc/metadata/how-to-delete-account/4.jpg)

---

### ⚠️ Remarques importantes

- **Action permanente** — Toutes vos données de jeu seront définitivement supprimées
- **Aucune récupération** — Cette action ne peut pas être annulée après la fin du compte à rebours
