# Hur Man Raderar Konto

Om du vill radera ditt konto från något **ZeroPlay** spel, följ dessa steg:

---

### Steg 1: Öppna Din Profil

Tryck på fliken *Me* i det nedre navigationsfältet för att komma åt din profilsida.

![Profile](https://dl.zeroplay.cc/metadata/how-to-delete-account/1.jpg)

### Steg 2: Hitta Raderingsalternativet

Scrolla ner till sektionen <span className="danger-text">DANGER ZONE</span> och tryck på knappen <span className="danger-text">Delete Account</span>.

![Delete Account](https://dl.zeroplay.cc/metadata/how-to-delete-account/2.jpg)

### Steg 3: Bekräfta Radering

Granska varningen och tryck på *Continue* för att fortsätta med kontoradering.

![Confirm Deletion](https://dl.zeroplay.cc/metadata/how-to-delete-account/3.jpg)

### Steg 4: Slutlig Nedräkning

Raderingsprocessen börjar med en 10-sekunders nedräkning. Du kan fortfarande avbryta genom att trycka på *Cancel Deletion* under denna tid.

![Account Deletion in Progress](https://dl.zeroplay.cc/metadata/how-to-delete-account/4.jpg)

---

### ⚠️ Viktiga Anmärkningar

- **Permanent Åtgärd** — All din speldata kommer att raderas permanent
- **Ingen Återställning** — Denna åtgärd kan inte ångras efter att nedräkningen är slut
