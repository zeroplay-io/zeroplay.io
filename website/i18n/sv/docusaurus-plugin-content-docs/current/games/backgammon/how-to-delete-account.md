# Hur Man Raderar Konto

Om du vill radera ditt konto från **Backgammon - Fair Board Games**, följ dessa steg:

---

### Steg 1: Öppna Inställningar

Starta spelet och tryck på knappen *Settings* i det övre högra hörnet.

![Settings](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/1.jpg#landscape)

### Steg 2: Hitta Raderingsalternativet

Scrolla till botten av inställningsfönstret och tryck på knappen <span className="danger-text">Delete Account</span>.

![Delete Account](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/2.jpg#landscape)

### Steg 3: Verifiera Din Åtgärd

Skriv in bekräftelsekoden som visas på skärmen och tryck sedan på knappen *Delete* för att fortsätta.

![Confirm Deletion](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/3.jpg#landscape)

### Steg 4: Slutlig Nedräkning

Raderingsprocessen börjar med en 10-sekunders nedräkning. Du kan fortfarande avbryta genom att trycka på *Cancel Deletion* under denna tid.

![Account Deleted](https://dl.zeroplay.cc/metadata/backgammon/how-to-delete-account/4.jpg#landscape)

---

### ⚠️ Viktiga Anmärkningar

- **Permanent Åtgärd** — All din speldata kommer att raderas permanent
- **Ingen Återställning** — Denna åtgärd kan inte ångras efter att nedräkningen är slut
